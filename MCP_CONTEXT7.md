# MCP Context7 Integration

## What is MCP Context7?

MCP (Model Context Protocol) Context7 is a tool that provides up-to-date documentation and code examples for programming libraries and frameworks.

## How This Project Uses Context7

This project is built with standard Next.js and React patterns. If you need to look up documentation or best practices, you can use the Context7 MCP tools.

## Available Context7 Tools

### 1. resolve-library-id
Finds the Context7-compatible library ID for a package.

**Example Usage:**
```
Query: "How to use zustand for state management?"
Library: "zustand"
```

### 2. query-docs
Retrieves documentation for a specific library and query.

**Example Usage:**
```
Library ID: "/pmndrs/zustand"
Query: "How to create a store with Zustand?"
```
